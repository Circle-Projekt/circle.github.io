# Circle – Die smarte Tauschhütte

> Studienprojekt · Progressive Web App · KI-gestützte Objekterkennung

Dieses Repository enthält die **Projektwebsite** von Circle, erreichbar unter:

**🌐 [circle-projekt.github.io/circle.github.io](https://circle-projekt.github.io/circle.github.io/)**

---

## Worum geht es bei Circle?

Tauschhütten sind eine großartige Idee aber niemand weiß, was gerade drin ist.
Circle löst das Problem:

1. **Fotografieren** – Objekt mit der Handykamera direkt in der PWA aufnehmen
2. **Klassifizieren** – eine KI erkennt das Objekt und ordnet es automatisch einer Kategorie zu
3. **Reinlegen** – das Objekt landet in der Hütte und gleichzeitig im Online-Inventar
4. **Finden & tauschen** – andere stöbern online und sehen, was bereitsteht, bevor sie hingehen

Circle entsteht als Uni-Studienprojekt und wird als **Progressive Web App (PWA)**
umgesetzt, läuft also direkt im Browser auf jedem Smartphone, ohne App-Store
und ohne Installation.

## Website lokal ansehen

Repo klonen und die `index.html` einfach im Browser öffnen:

```bash
git clone https://github.com/Circle-Projekt/circle.github.io.git
cd circle.github.io
# Datei im Browser öffnen, z. B.:
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```
